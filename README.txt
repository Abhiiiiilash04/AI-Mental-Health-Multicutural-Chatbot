# ZenMate AI Mental Health Chatbot

## Features
- Uses OpenAI GPT-3.5 to provide human-like empathetic responses
- Culturally adaptive responses based on language
- Supports multilingual input (e.g., Swahili & English)
- Lightweight and easy to run

## Setup Instructions

1. Make sure you have Python 3.8+ installed
2. Install required packages:
   pip install -r requirements.txt

3. Download spaCy English model:
   python -m spacy download en_core_web_sm

4. Create a `.env` file with your OpenAI API key:
   OPENAI_API_KEY=sk-...

5. Run the app:
   python app.py

6. Open your browser:
   http://localhost:5000
